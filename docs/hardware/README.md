# Hardware Documentation

Store all hardware-related documentation here, including:

- **Schematics**: Wiring diagrams and circuit layouts.
- **Datasheets**: Specifications for servos, microcontrollers (ESP32), sensors (IMU, Camera), and batteries.
- **CAD Models**: 3D designs for chassis and custom parts.
- **Bill of Materials (BOM)**: Component lists and sourcing information.
